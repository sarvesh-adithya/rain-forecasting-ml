# 🌧️ Rain Forecasting using Machine Learning

## 📌 Problem Statement
The objective of this project is to predict whether it will rain based on weather conditions such as temperature, humidity, and wind.

---

## 📊 Dataset
- Weather dataset with multiple features  
- Includes temperature, humidity, wind, and pressure  
- Target: Rain (Yes/No)

---

## ⚙️ Approach
- Data cleaning and preprocessing  
- Handling missing values  
- Feature selection  
- Applied classification models:
  - Logistic Regression  
  - Random Forest  

---

## 📈 Results
- Achieved ~88% accuracy  
- Improved model performance through tuning  

---

## 🛠️ Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-learn  

---

## 💡 Key Takeaways
- Built classification model for weather prediction  
- Improved understanding of feature engineering
